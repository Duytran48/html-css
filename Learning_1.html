The basic Language of the Web:HTML

Posted by Laura Jones on Monday,June 21st 2028

All modern websites and weblication are built using three fundamentals technologies: HTML,CSS,JavaScipt.There are the languages of the Web.

In the post, let's focus on HTML. We will learn what HTML is all about, and why you should too learn it.

What is HTML ?

HTML stand for Hyper Text Markup Language.It's a markup language for developers use to structure and describe the content of a web page (not a programing language).

HTML consists of elements that describe different types of contents : paragraph,links,images,heading,videos,etc.Web browser understand HTML and render HTML code as a websites.

In HTMl,each element is made up of 3 parts:

  1. The opening tags
  2. The closing tags
  3. The actual elements

You can learn more at the MDN Web Docs

Why should you learn HTML ?

There are countless reasons for learning the fundamental language of the Web. Here are 5 of them:

  • To be able to use the fundamental web dev language
  • To hand-craft beautiful websites instead of relying on tools like WordPress or Wix
  • To build web applications
  • To impress friends
  • To have fun
